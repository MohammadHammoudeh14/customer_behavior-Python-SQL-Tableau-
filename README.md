# Customer Behavior Analysis 🛒

## 📌 Project Overview
This project addresses the strategic business question: **"How can the company leverage consumer shopping data to identify and optimize marketing and product strategies?"** The analysis follows a full data science lifecycle: from raw data cleaning and SQL deep-diving to interactive visualization and strategic business recommendations.

## 📁 Repository Structure
The project is organized as follows:
* **`customer_shopping_trends.ipynb`**: Python notebook (Google Colab) used for data cleaning and initial EDA.
* **`customer_shopping_behavior.csv`**: The original raw dataset.
* **`customer_behavior_final.csv`**: The cleaned and processed dataset after Python manipulation.
* **`Customer_behavior.sql`**: Advanced SQL queries used for behavioral analysis.
* **`customer behavior.twb`**: The Tableau workbook containing the interactive dashboards.
* **`Customer behavior.png`**: A high-resolution screenshot of the final Tableau dashboard.

## 🛠 Tech Stack & Workflow

### 1. Python (Data Preprocessing)
* Used for data manipulation and cleaning.
* Handled missing values (nulls) and dropped redundant columns.
* **Feature Engineering:** Created the `age-group` column to segment customers for better targeting.

### 2. SQL (Advanced Analysis)
* Performed complex queries to extract trends in purchasing frequency, revenue per segment, and shipping preferences.

### 3. Tableau (Visualization)
* Transformed data into actionable insights through an interactive dashboard.
* Visualized KPIs related to customer demographics and sales performance.

## 💡 Key Business Recommendations
Based on the analysis, the following strategies are proposed:
* **Boost Subscription:** Promote exclusive benefits to non-subscribers to increase retention.
* **Customer Loyalty Programs:** Implement reward systems for frequent buyers to migrate them to the "Loyal" segment.
* **Review Discount Policy:** Strategically balance discount-driven sales boosts with margin control.
* **Product Positioning:** Highlight top-rated and best-selling products in marketing campaigns.
* **Target Marketing:** Prioritize efforts on high-revenue age groups and express-shipping users.
